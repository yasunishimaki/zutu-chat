# 頭痛博士アプリ（zutu-chat）

頭痛専門医による解説動画60本と、その文字起こしデータを活用した、一般向けの頭痛情報Webアプリです。

利用者が普段の言葉で質問すると、博士キャラクター「レイン博士」が専門医の解説から関係する内容を探し、分かりやすく案内します。回答の根拠となった動画をその場で視聴できます。

> 頭痛博士は、頭痛専門医の解説動画を案内するキャラクターです。診断や治療の判断は行いません。

## 技術構成（予定）

| 技術 | 用途 |
|---|---|
| React + Cloudflare Pages | アプリ画面の公開 |
| Cloudflare Worker | APIキー保護・安全確認・中継 |
| Workers KV | 定番質問の回答キャッシュ |
| OpenAI File Search / Responses API | 文字起こし検索と回答生成 |
| OpenAI STT | 音声入力の文字化 |
| Cloudflare D1 | 動画・医師情報・公開状態の台帳 |
| Cloudflare R2 | 文字起こし原本の保管 |
| YouTube（限定公開） | 動画の配信 |
